Здесь нужно описать основные вещи, как работает админка, основные библиотека и классы. 

Контроллеры:

	Admin/Ajax
		-Ajax ответы сервера, в основном юзается для JS скриптов
		
	Admin/Frontpage
		-Главная страница админки
		
	Admin/Login
		-Авторизация пользоввателя в админке
		
	Admin/Users
		-Пользователи, добавление, удаление, редактирование, история изменений
		
	Admin/Tables
		-Таблицы
		
	Admin/Sections
		-Разделы
		
	Admin/Reviews
		-Обзоры
		
	Admin/Reviews_Articles
		-Статьи обзоров
		
	Admin/Reviews_Cases
		-Кейсы обзоров
		
	Admin/Reviews_Tables
		-Таблицы обзоров
		
	Admin/Reviews_Interviews
		-Интервью обзоров
		

Модели:

	Model_Reviews
		-Модель обзоров, выборки,удаления, обновления таблиц связанных с обзорами, статьями обзоров и тд
	
	Model_Tables
		-Модель таблиц
		
		
Хелперы:
	
	Common_Helper
		-Основной класс помошник, выполняет разлычные действия с различными сущностями, объектами.
	
	Image_Helper
		-Класс-помошник для работы с картинками
		
	My_Auth_Helper
		-Статический класс http авторизации, сейчас не юзается.
		
	My_Entities_Helper
		-Статический класс для работы с подключенными сущностями, например Таблицы подключенные в статьях или интервью...
		
	My_Files_Helper
		-Стат.класс для работы с файлами
		::set_table_dirs - создаем иерархию папок на сервере для загрузки xls файлов таблиц
		
	My_Url_Helper
		-Стат.класс для работы с урлами, редиректы и т.д.
		

Либы:
	
	Admin_Menu
		-Класс для работы с меню админки, генерация, удаления не доступных разделов и т.д.
		
	Breadcrumbs_Block
		-Крошки, вся работа с крошками в админке
		
	ExcelReader
		-Не используется
	
	ExcelToTable
		-Парсинг *.xls файлов и генерация таблиц
		
	Generate_Table
		-Генерация пустой таблицы
		
	Header_Block
		-Шапка сайта, необходимый функционал
		
	Image_Easy
		-Легкий ресайз картинок, пока не юзается
		
	My_History
		-История именений в админке, описывает весь необходимый функционал модуля
		
	My_Users
		-Пользователи, описывает функционал для работы пользователя и разделя пользователи
		
	Reviews_Collector
		-Класс сборщик POST и GET для обзоров и передача в модель
		
	Reviews_Views
		- Вьюха обзоров, описывает физический функционал для вывода данных, связывает модель-контроллер
		
	TableCollector
		- Класс сборщик post данных для модуля таблицы
		
	Tables_Views
		- Вьюха для модуля таблиц
		
	Validate
		- Валидатор, описывает необходимые методы валидации данных
